# Accomodation_V2
### Used technologies:
- .Net 7
- C# 10
- EF Core 7
- TSQL
## First setup:
1. Open the DB-scripts folder
1. Run the create-local-db.bat file
1. Run the script inside the Generate-db-script.sql file
1. Run the script inside the Create_dml_triggers.sql file
